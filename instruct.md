# 🎯 OBJECTIF DU PROJET
Créer un portfolio web professionnel pour un étudiant en informatique, avec une image moderne, crédible et orientée vers l'emploi et les opportunités.

Le site doit :
- Présenter clairement le profil
- Mettre en avant les compétences techniques
- Montrer les projets réalisés
- Permettre un contact facile
- Être responsive (mobile + desktop)

# 🧱 STRUCTURE DU SITE

## 1. HERO SECTION (Accueil)
Contenu :
- Nom + prénom
- Titre : "Développeur Web & Étudiant"
- Une phrase d'accroche (ex : "Je crée des applications modernes et performantes")
- Boutons :
  - Voir mes projets
  - Me contacter

Option :
- Photo professionnelle (ou avatar moderne)

---

## 2. À PROPOS
Contenu :
- Présentation courte (qui je suis)
- Parcours académique
- Motivation / objectifs

Exemple attendu :
"Je suis étudiant passionné par le développement web, avec un intérêt particulier pour..."

---

## 3. COMPÉTENCES

Organiser en catégories :

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- C#
- .NET

### Outils
- Git
- VS Code
- Visual Studio

Afficher :
- Sous forme de badges ou barres de progression

---

## 4. PROJETS

Afficher sous forme de cartes :

Pour chaque projet :
- Nom du projet
- Description courte
- Technologies utilisées
- Image (optionnel)
- Lien GitHub ou démo

Exemples de projets :
- Application de gestion de stock
- Programme C# console
- Mini site web

---

## 5. CONTACT

- Email

## 6. FOOTER
