# 🎨 DESIGN SYSTEM - PORTFOLIO

---

# 🎯 OBJECTIF DESIGN
Créer une interface moderne, professionnelle et minimaliste inspirée des développeurs juniors internationaux.

Doit inspirer :
- Confiance
- Clarté
- Compétence

# 🎨 PALETTE DE COULEURS

## Couleurs principales
- Primary : #4F46E5 (bleu moderne)
- Secondary : #0F172A (bleu nuit)
- Background : #FFFFFF (clair)
- Dark mode background : #020617

## Couleurs secondaires
- Accent : #22C55E (vert succès)
- Error : #EF4444 (rouge)

- Text secondary : #6B7280

# 🔤 TYPOGRAPHIE

Police principale :
- Inter
- Poppins (alternative)

Hiérarchie :
- H1 : 40px - bold
- H2 : 28px - semi-bold
- H3 : 22px
- Texte : 16px

---

# 🧱 LAYOUT

Grille :
- Max width : 1200px
- Margin auto
- Padding : 20px

Sections espacées :
- 80px entre sections

---

# 🧭 NAVBAR

- Fixe en haut
- Transparent → devient solide au scroll
- Logo à gauche
- Menu à droite :
  - Accueil
  - À propos
  - Compétences
  - Projets
  - Contact

Mobile :
- Menu hamburger

---

# 🖼 HERO SECTION

Disposition :
- Texte à gauche
- Image à droite

Contenu :
- Nom (gros titre)
- Titre métier
- Phrase d’accroche
- 2 boutons :
  - Voir projets (primary)
  - Contact (outline)

Effets :
- Fade-in au chargement
- Animation légère sur bouton hover
